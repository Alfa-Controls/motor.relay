typedef struct SENSOR_TAG Sensor

Sensor structure definition.

It provide properties and methods to handle with physical sensors used to protect a electrical motor. These sensors have two protective regions: Instantaneous trip and temporized trip. In instantaneous monitoring, a trip event will be generated after a sensor raw value get higher than the trigger level. In temporized monitoring, it will wait a specified time after reach the trigger level before start a trip.
This structure methods requires TimeSystem.h to provide the currentTimeMillis.
