This project is motor protection system. It will read some sensors and create trip signals to stop the motor, based on two different protection regions\+: instantaneous and Temporized.\hypertarget{index_Methodology}{}\doxysection{Methodology}\label{index_Methodology}
The project uses the MCC to create the first hardware access layer. This layer is isolated from the rest of the project by the application control (\mbox{\hyperlink{app_8h}{app.\+h}}/app.c files).~\newline
 In the application control, the hardware layer are used to provide data to the abstraction and business layers. In this approach only the \mbox{\hyperlink{app_8c}{app.\+c}} file has access to hardware and business, keeping all project encapsulated with well defined responsibilities for every file.\hypertarget{index_Application}{}\doxysection{Modules\+:}\label{index_Application}
